# SQL CRUD Operations Assignment

## Description
This repository contains SQL queries for performing **CRUD operations** on a `students` table using the **Supabase SQL Editor**.  
The assignment demonstrates:

- Creating a table
- Inserting multiple records
- Reading data using SELECT queries
- Updating records
- Deleting records

All queries are written in SQL only, **without using any backend code** or Supabase Table Editor UI.

---

## Table Structure

| Column Name | Data Type | Description                  |
|------------|-----------|------------------------------|
| id         | SERIAL    | Unique student ID (Primary Key) |
| name       | TEXT      | Student name                 |
| email      | TEXT      | Student email                |
| age        | INTEGER   | Student age                  |
| course     | TEXT      | Course enrolled              |
| created_at | TIMESTAMP | Record creation time         |

---

## Queries Included

1. **Create Table**  
2. **Insert Data** (7 sample student records)  
3. **Read Data** (SELECT queries)  
4. **Update Data**  
5. **Delete Data**

All queries are available in the [`queries.txt`](./queries.txt) file.

---

## Usage
1. Open your **Supabase project**.  
2. Go to **SQL Editor**.  
3. Copy the queries from `queries.txt` and run them one by one.  
4. Observe the results for each operation.
